# 📚 Repositório de Estudos Web  

Este repositório reúne meus estudos práticos e teóricos sobre desenvolvimento web, abordando desde os fundamentos da comunicação cliente-servidor até a integração com banco de dados utilizando PHP.

O objetivo é consolidar conhecimento, aplicar conceitos na prática e servir como base para projetos futuros.

---

## 🌐 O Ambiente Web: Cliente x Servidor  

Nesta seção, são explorados os conceitos fundamentais da arquitetura web:

- Diferença entre **cliente** (navegador) e **servidor**
- Como funciona uma requisição HTTP
- Ciclo de comunicação entre front-end e back-end
- Protocolos utilizados (HTTP/HTTPS)
- Estrutura básica de aplicações web

Esse entendimento é essencial para compreender como as aplicações funcionam na internet.

---

## 🏗️ Linguagem de Marcação de Hypertexto - HTML  

O HTML (HyperText Markup Language) é a base da construção de páginas web.

Conteúdos abordados:

- Estrutura básica de uma página HTML
- Tags principais (`<html>`, `<head>`, `<body>`)
- Elementos de texto, links, imagens e listas
- Formulários e entrada de dados
- Semântica HTML (uso correto das tags)

---

## 🎨 Linguagem de Marcação e Estilos - CSS  

O CSS (Cascading Style Sheets) é responsável pela estilização das páginas.

Principais tópicos:

- Seletores e propriedades
- Cores, fontes e espaçamentos
- Box Model
- Layouts com Flexbox e Grid
- Responsividade e design adaptável

---

## ⚡ Linguagem JavaScript  

O JavaScript permite adicionar interatividade às páginas web.

Conteúdos estudados:

- Variáveis, tipos de dados e operadores
- Estruturas condicionais e de repetição
- Funções
- Manipulação do DOM
- Eventos
- Integração com HTML e CSS

---

## 🔗 Integração do PHP com Banco de Dados  

O PHP é utilizado para o desenvolvimento back-end e integração com banco de dados.

Tópicos abordados:

- Sintaxe básica do PHP
- Processamento de formulários
- Conexão com banco de dados (ex: PostgreSQL/MySQL)
- CRUD (Create, Read, Update, Delete)
- Organização de código back-end
- Segurança básica (validação de dados)
